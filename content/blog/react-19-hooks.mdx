# React 19 Hooks Explained

React 19 introduces several new hooks and patterns. Let's explore the most impactful ones.

## The `use` Hook

The new `use` hook is a game-changer for data fetching and context consumption.

### Reading Promises

You can now read promises directly in render:

```tsx
function Comments({ commentsPromise }) {
  const comments = use(commentsPromise)
  
  return (
    <ul>
      {comments.map(comment => (
        <li key={comment.id}>{comment.text}</li>
      ))}
    </ul>
  )
}
```

### Reading Context Conditionally

Unlike `useContext`, the `use` hook can be called conditionally:

```tsx
function Theme({ children }) {
  if (shouldUseTheme) {
    const theme = use(ThemeContext)
    return <div style={{ color: theme.color }}>{children}</div>
  }
  return children
}
```

## Actions and Transitions

### The `useActionState` Hook

Manage form state with the new actions pattern:

```tsx
function Form() {
  const [state, formAction] = useActionState(submitForm, initialState)
  
  return (
    <form action={formAction}>
      {state.error && <p>{state.error}</p>}
      <input name="email" />
      <button>Submit</button>
    </form>
  )
}
```

### The `useOptimistic` Hook

Implement optimistic updates easily:

```tsx
function Messages({ messages }) {
  const [optimisticMessages, addOptimistic] = useOptimistic(messages)
  
  async function sendMessage(formData) {
    addOptimistic({ text: formData.get('message'), pending: true })
    await deliverMessage(formData)
  }
  
  return (
    <ul>
      {optimisticMessages.map(m => (
        <li key={m.id} style={{ opacity: m.pending ? 0.5 : 1 }}>
          {m.text}
        </li>
      ))}
    </ul>
  )
}
```

## Best Practices

1. **Use Suspense boundaries** - Wrap components that use promises
2. **Handle errors** - Use Error Boundaries for failures
3. **Consider server components** - Fetch data on the server when possible

## Conclusion

React 19's new hooks simplify common patterns and enable more powerful compositions. Start experimenting with these features today!
